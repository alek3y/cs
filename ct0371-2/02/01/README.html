Computer Science

Array

La struttura dati può essere implementata con un array P contenente tutti i nodi.

In entrambe le rappresentazioni, avendo n nodi, la complessità spaziale è data da S(n)=Θ(n).

Un albero di esempio può essere:

Padri

In questa rappresentazione l'array è composto da coppie (info, parent).

Nell'esempio, l'array P sarà rappresentato come P = [(A, -1), (B, 0), (C, 0), (D, 1), (E, 1)].

Implementazione

  • Padre

    padre(Tree P, Node v) -> Node | NIL
  if P[v].parent == -1
    return NIL
  else
    return P[v].parent

    per cui T(n)=Θ(1).

  • Figli

    figli(Tree P, Node v) -> [Node]
  l = []
  for i = 0 to P.length-1
    if P[i].parent == v
      push(l, i)  // Assunto 𝛩(1)
  return l

    per cui T(n)=Θ(n).

Posizionale

In questo caso la rappresentazione è applicabile per alberi k-ari completi con k≥2 salvato in P.k. Solamente info è salvato nell'array, perchè la relazione tra i nodi è conservata come indice nell'array.

Nell'esempio, l'array P sarà rappresentato come P = [A, B, C, D, E], ovvero:

La radice è in posizione 0, mentre i figli del nodo v sono in posizione k⋅v+1+i per i=0,...,k−1. Definiamo v padre di f quando si ricava che: ​⇔k⋅v+1≤f≤k⋅v+1+k−1⇔k⋅v≤f−1≤k⋅v+k−1⇔v≤kf−1​≤v+<1kk−1​​​⇔v≤kf−1​<v+1⇒v=⌊kf−1​⌋​ dato che v∈N.

Implementazione

  • Padre

    padre(Tree P, Node v) -> Node | NIL
  if v == 0
    return NIL
  else
    return floor((v - 1)/P.k)

    per cui T(n)=Θ(1).

  • Figli

    figli(Tree P, Node v) -> [Node]
  l = []
  if P.k*v + 1 >= P.length
    return l
  else
    for i = 0 to P.k-1
      push(l, P.k*v + 1 + i)  // Assunto 𝛩(1)
    return l

    per cui T(n)=O(k) perchè nel caso migliore è Θ(1).
