Array

La struttura può essere un array A di n elementi ordinati dal campo key e contenenti anche il campo info.

La complessità spaziale sarà quindi S(n)=Θ(n).

Implementazione

  • Search

    Prima di search sarà necessario implementare search_index che fa la ricerca binaria:

    search_index(Dizionario A, Chiave K, int p, int r) -> int
  if p > r
    return -1             // 𝛩(1)
  else
    med = floor((p+r)/2)  // 𝛩(1)
    if A[med].key == k
      return med          // 𝛩(1)
    else
      if A[med].key > k
        return search_index(A, K, p, med-1)  // T(n/2)
      else
        return search_index(A, K, med+1, r)  // T(n/2)

    che per l'else risulterà essere una ricorrenza: T(n)={Θ(1)T(2n​)+Θ(1)​se n=0se n>0​ da cui T(n)=Θ(logn) per il teorema master.

    Di conseguenza per search basterà:

    search(Dizionario A, Chiave K) -> Elem | NIL
  i = search_index(A, K, 1, A.length)  // 𝛩(log n)
  if i == -1
    return NIL        // 𝛩(1)
  else
    return A[i].info  // 𝛩(1)

    da cui T(n)=Θ(logn)+Θ(1).

  • Insert

    insert(Dizionario A, Chiave K, Elem V)
  i = 1
  while i <= A.length and A[i].key < K  // d*i volte, con massimo i = A.length
   i++

  if i <= A.length and A[i].key == K
    A[i].info = V
  else  // Più costoso del corpo dell'if
    reallocate(A, A.length + 1)   // O(n)
    A.length++                    // 𝛩(1)

    for j = A.length down to i+1  // (n+1)-(i+1)+1 = n-i+1 volte, dove n+1 per A.length++
      A[j] = A[j-1]               // d*1

    A[i].key = K                  // 𝛩(1)
    A[i].info = V                 // 𝛩(1)

    dove O(n) per la reallocate perchè al massimo copierà n elementi in un'altra area di memoria.

    Di conseguenza, T(n)=Θ(1)+d⋅i+O(n)+(n−i​+1)⋅d=O(n).

  • Delete

    delete(Dizionario A, Chiave K)
  i = search_index(A, K, 1, A.length)  // 𝛩(log n)
  for j = i to A.length-1              // (n-1)-i+1 = n-1 volte
    A[j] = A[j+1]                      // d*1

  reallocate(A, A.length-1)            // O(n)
  A.length--                           // 𝛩(1)

    per cui T(n)=Θ(logn)+(≤nn−i​​)⋅d+O(n)=O(n).

  • Reallocate

    La reallocate può raddoppiare e dimezzare la capacità h di un array con n elementi, secondo: n≤h<4n, ∀n>0⇕n≤h ∧ n>4h​⇕4h​<n≤h iniziando con n=0 e h=1, se n>h allora h viene raddoppiato e se n≤4h​ viene dimezzato.

    Di conseguenza S(n)=Θ(h)=Θ(n) perchè h<4n.

Analisi ammortizzata

Tramite l'analisi ammortizzata è possibile analizzare il costo medio di n operazioni.

Per esempio, si vogliono fare n inserimenti su un array con 0 elementi di capacità 1, allora: Ci​={i1​se ∃k:i=2k+1altrimenti​ con Ci​ che rappresenta il costo dell'i-esimo insert, che vale i quando la reallocate raddoppia la capacità.

Il costo per n inserimenti risulterà quindi essere: C(n)=i=1∑n​Ci​​≤n+i=0∑⌊log2​n⌋​2i≤n+2−12⌊log2​n⌋+1−1​≤3n​ dove i=0∑⌊log2​n⌋​2i rappresenta la somma di tutti gli i-esimi inserimenti che hanno costo i.

Di conseguenza il costo ammortizzato sarà nC(n)​≤n3n​=3=Θ(1).
